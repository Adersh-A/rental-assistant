# Rental Assistant Using AI
A Spring Boot application that extracts rental listings from WhatsApp group exports using AI.

## Overview

This project automates the process of finding rental properties from WhatsApp group chats.

Instead of manually searching through hundreds or thousands of messages, users can upload a WhatsApp chat export via Telegram. The application parses the messages, stores them, uses an LLM to identify rental listings, and exposes structured data for searching and recommendations.
